## local setup

Sensitive values must stay in a local-only `Secrets.xcconfig` file.

- `Secrets.xcconfig` is ignored by git and should never be committed.
- Copy `Secrets.example.xcconfig` to `Secrets.xcconfig` and fill in your own values before running the app.
- Local Amplify config files are also kept out of git in this repo.
- Keep `AmplifyConfig/amplifyconfiguration.json` and `AmplifyConfig/awsconfiguration.json` local-only.
